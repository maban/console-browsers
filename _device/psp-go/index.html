The screen is not a touch screen. There is a small cursor that can be moved around using the left thumbstick. The d-pad cycles through links on the page and scrolls.

The left and right shoulder pads move back and forward through the browser history. Holding down the square button and tapping the right shoulder pad opens up a new tab. There can be up to 3 tabs open at once.

The browser standard is very low. A lot of sites don't load, or content doesn't appear. JavaScript is enabled but is very unreliable. It does not handle HTTPS very well coming up with frequent warnings, even on sites like Google.

Text size can be adjusted in the settings. It has three display modes: Normal (width is not adjusted), Just-Fit (width is adjusted to match screen width) and Smart-Fit (content is reordered so page width matches screen width).
